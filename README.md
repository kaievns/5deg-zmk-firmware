# 5deg ZMK Firmware

This is custom ZMK firmware for my new keyboard. It's a fully custom build with
bespoke PCBs so it will unlikely apply to any normal nice!nano based keyboards.

What it has:

- Plan-B PCB shield configurations for the 5deg keyboard
- Custom pmw3610 optical sensor driver
- Custom input processors for mouse cursor acceleration and smoothing
- Custom temp layer behaviour
- My regular 33% keyboard keymap
